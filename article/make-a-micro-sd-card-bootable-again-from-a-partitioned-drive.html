Make a Micro SD Card bootable again from a partitioned drive


person Sebastian Doe

timer 25 December 2019



This method allows you to make an SD card bootable again by just typing out 10 lines of code in CMD (in Windows).

This completely wipes the data off the SD card so be careful when completing the steps.

diskpart
list disk
select disk {number of the disk you want to format from the above line}
clean
create partition primary
select partition 1
active
format fs=fat32 quick
assign
exit

That's it! Your SD card should be good as new.
